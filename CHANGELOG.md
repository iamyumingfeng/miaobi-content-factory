# 变更日志

本项目的所有重要更改都将记录在此文件中。

## [1.0.0] - 2026-06-20

### 新增
- 🎉 **核心功能**
  - 用户认证系统（账号密码登录）
  - JWT 双 Token 机制（access_token + refresh_token）
  - 三表分离用户管理（super_admin / operator / sub_user）
  - 四级层次结构（平台 → 分类 → 标签 → 项目）

- 📝 **模板管理**
  - 模板平台、分类、标签的 CRUD
  - Prompt 模板可视化编辑
  - 动态变量占位符配置
  - 模板复制与版本管理

- 🎨 **素材管理**
  - 多类型素材上传（文本/图片/视频）
  - 双素材库类型（创作库 / 对标库）
  - 素材标签分类
  - 素材收藏功能
  - COS 对象存储集成

- 🤖 **AIGC 内容批量生成**
  - 8 个模型平台支持（百炼、火山引擎、即梦、可灵、AutoGLM、月之暗面、DeepSeek、灵牙AI）
  - 双认证方式（标准 API Key + AccessKey/SecretKey）
  - 并发控制与自动限流
  - 失败自动重试（指数退避）
  - WebSocket 实时进度推送
  - 子任务暂停/继续/重试
  - 智能去重检测（文案 + 图片）
  - 对标素材参考生成

- 📤 **内容分发**
  - 批量分发至创作者
  - 分发状态跟踪
  - 创作者一键复制/下载
  - 发布确认机制

- 📊 **仪表盘**
  - 统计卡片展示
  - 趋势图分析
  - 最近任务列表
  - 失败告警提示

- ⚙️ **系统设置**
  - 个人设置（自定义昵称、修改密码）
  - 模型配置管理
  - 用户默认模型设置

### 技术架构
- 🏗️ **后端**
  - Python 3.11+ / FastAPI 框架
  - SQLAlchemy 2.0 ORM + Alembic 数据迁移
  - Celery + Redis 异步任务队列
  - WebSocket 实时通信
  - 三阶段事务架构（解决连接池耗尽问题）

- 🎨 **前端**
  - Vue 3 + TypeScript + Vite
  - Element Plus UI 组件库
  - Pinia 状态管理
  - Axios HTTP 客户端

- 📱 **小程序**
  - 原生微信小程序框架
  - TDesign UI 组件库
  - 微信授权登录

- 🗄️ **数据存储**
  - MySQL 8.0 关系型数据库
  - Redis 7.0 缓存和消息队列
  - 腾讯云 COS 对象存储

- 🐳 **部署**
  - Docker + Docker Compose 容器化
  - Nginx 反向代理
  - 自动化部署脚本

### 性能优化
- ⚡ 三阶段事务架构：连接持有时间从 100秒降至 <2秒
- ⚡ 数据库连接池优化：pool_size=50, max_overflow=30
- ⚡ 向量缓存：content_embedding 缓存嵌入向量
- ⚡ 批量查询优化：generation_task 冗余进度字段

### 安全特性
- 🔐 JWT 双 Token 认证机制
- 🔐 登录失败锁定（5次失败锁定15分钟）
- 🔐 数据隔离：owner_operator_id 多租户隔离
- 🔐 密码哈希存储（bcrypt）
- 🔐 CORS 跨域配置
- 🔐 SQL 注入防护（参数化查询）
